军训之声

军训的生活早已告一段落，但是它带给我的记忆不会轻易抹去，我要来仔细的倾听军训的声音，回味军训的生活,感谢它十五天的陪伴

军训的生活是枯燥乏味的，从第一天开始，不想军训的我就开始满是怨言，尤其当我知道还有晚训，早训这样的恐怖如斯的东西以后，我就濒临绝望的边缘

可当过了一段时间以后，我便开始爱上了这种节奏，爱上了这种有些枯燥的生活

正如歌里唱的那样，这种生活有苦有乐，丰富多彩
